School Of Computing web server

The university has a webserver setup for students use. You should be able to access it with your student credentials used on any of the university systems.

The base address is http://socweb8.napier.ac.uk/ and you will find both the documentation and a way to create your own database there.

When creating your database, please store the information displayed for reuse in the future when setting up your projects.

Access the server from the university

socweb8.napier.ac.uk can be accessed from the university machines on the I: drive. If the drive is not mounted on your session, please use Start/All Programs/Application/Faculty of Engineering/School of Computing/MapUnixDrive and follow the procedure. If something goes wrong, please ask support to the helpdesk.

Once your I: drive is mounted, you can access the files contained in your I:\public_html directory at the address http://socweb8.napier.ac.uk/~matriculation, matriculation being your actual matriculation number.

Access the server from your own computer

There are two easy ways to use the server from your machines, both using SSH in different ways.

If you are using Linux or Mac, you can use the terminal and connect using the command ssh matriculation@socweb8.napier.ac.uk. You will then have a command line access to the server.

Another way would be using FileZilla or WinSCP and manage your files with it, on port 22, host socweb8.napier.ac.uk and your university credentials.

Remote desktop

Go to https://desktop.napier.ac.uk/ and download VMWare Horizon Client.

VMWare will ask you for a Connection Server, use https://desktop.napier.ac.uk.

Use your matriculation number and usual univeristy password to login.

Everything is temporary on this session, so please work or backup your code either in I:\ or H:\.
